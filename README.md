# GlowUp AI Salon Rating

AI-powered salon look rating system built using React + Vite + Gemini AI.

## Features
- Upload a face photo
- Get AI rating out of 5
- Personalized grooming & beauty suggestions
- Before vs After comparison

## Tech Stack
- React 18
- Vite
- Gemini AI

## Setup
1. npm install
2. create .env.local with VITE_GEMINI_API_KEY
3. npm run dev
